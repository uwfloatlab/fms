This distribution contains the source code for the Float Mission Simulator
used to simulate, debug, and test float firmware.  This distribution is
self-contained, (almost) complete, and should build/run under most recent
versions of linux.  It requires that the FLEX package be installed on the host
linuc computer.  

To build the executable fms, fmsRossSea, fmsRossSeaIc utility, execute
'make clean' and then 'make'.  If the build is successful, the fms
utility will be found in the bin directory.
